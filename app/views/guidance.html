Guidance:

This service enables you to send an email message to multiple email accounts.

Only send messages to people who have signed up to receive notifications from the FSA. If you have any queries check with GDPR team. Policy notice

To send a message:

  • Upload a list of email addresses. The file must be in csv format. All email addresses should be entered in the first column only. Do not add a header row.
  • Type your email subject in the form below and click ‘Send’. It will be sent immediately.

Delivery statistics, showing messages sent in the last 7 days, are available on the dashboard.

The list of email addresses is removed from the app automatically. It will be on the server for a maximum of 2 hours.



Message formatting:

The form supports basic markdown to add text formatting.

To add a title, use a hash:

# This is a title


To add bullet points, use asterisks:

* point 1
* point 2
* point 3


Seperate paragraphs by three dashes:

First paragraph
---
Second paragraph
